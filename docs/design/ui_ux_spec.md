|------|-----|

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 방 선택 | 56×56dp | 탭 |
| 카드 선택 | 100×140dp | 탭 |
| 카드 상세 | 카드 위 | 롱프레스 |
| 적 정보 | 적 영역 | 탭/롱프레스 |

### 터치 피드백
- 방 선택: 확대 + 경로 하이라이트
- 카드 선택: 카드 상승 애니메이션
- 카드 사용: 발동 이펙트

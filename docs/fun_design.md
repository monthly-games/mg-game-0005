## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 턴제 퍼즐/전투 진행                                  │
│         ↓                                               │
│  2. 층별 보상 선택 (스킬, 유물, 자원)                    │
│         ↓                                               │
│  3. 사망 시 런 종료 → 메타 보상 획득                     │
│         ↓                                               │
│  4. 메타 업그레이드/해금 후 재도전 ────────→ (1로 반복)   │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **턴제 퍼즐/전투**: 층별 퍼즐을 풀며 던전 진행
2. **층별 보상 선택**: 스킬/유물/자원 중 전략적 선택
3. **런 종료 → 메타 보상**: 사망 또는 클리어 시 영구 자원 획득
4. **메타 업그레이드**: 다음 런에 영향을 주는 영구 성장

## 3. 재미 축 (Fun Pillars)

### 3.1 빌드/조합 (Build)
- **목표**: 매 런마다 다른 빌드를 만드는 재미
- **핵심 경험**: 이번 런은 이 빌드로 가보자는 전략적 결정
- **주요 요소**:
  - 스킬 조합 시스템
  - 유물 시너지
  - 속성/테마 빌드
  - 선택지 분기

### 3.2 도전/재도전 (Challenge/Retry)
- **목표**: 실패해도 다시 도전하고 싶은 설계
- **핵심 경험**: 다음엔 더 잘할 수 있다는 확신
- **주요 요소**:
  - 점진적 난이도
  - 메타 progression
  - 업적 시스템
  - 랭킹/기록 경신

### 3.3 발견/학습 (Discovery)
- **목표**: 새로운 조합과 전략을 발견하는 재미
- **핵심 경험**: 이런 조합이 되네! 하는 발견의 기쁨
- **주요 요소**:
  - 숨겨진 시너지
  - 희귀 유물/스킬
  - 비밀 방/이벤트
  - 도감 시스템

## 4. 디자인 레버 (Design Levers)

### 4.1 테마 빌드 유도
| 레버 | 설명 |
|------|------|
| 강한 테마 빌드 | 각 런마다 강한 테마 빌드(독, 크리, 소환 등)를 형성하게 선택지 설계 |
| 구현 방향 | 첫 선택에서 방향 제시, 이후 선택지에서 관련 옵션 확률 증가 |

### 4.2 사망 리포트 시스템
| 레버 | 설명 |
|------|------|
| 학습 유도 리포트 | 사망 시 "이번 런에서의 중요한 지표/실수"를 리포트해 학습 유도 |
| 구현 방향 | "받은 독 데미지: 340 (전체 피해의 45%)", "버린 유물 중 시너지 가능: 2개" |

### 4.3 메타 Progression
| 레버 | 설명 |
|------|------|
| 반복 플레이 동기 | 메타 progression(새 유물/적/스테이지)으로 반복 플레이 동기 제공 |
| 구현 방향 | 10회 플레이 → 새 캐릭터 해금, 보스 처치 → 새 유물 풀 추가 |

### 4.4 후회 설계
| 레버 | 설명 |
|------|------|
| 정보 노출/패턴 설계 | 퍼즐 정보 노출/패턴 설계를 통해 "후회가 남는 선택"이 생기도록 조정 |
| 구현 방향 | 선택 전 다음 층 보스 정보 일부 공개, 선택 후 "그 유물이 있었으면..." 상황 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 런 당 플레이 시간 | 15~25분 | 런 시작 → 종료 시간 |
| 일일 런 횟수 | 3회 이상 | 런 시작 로그 |
| 재도전률 | 85% 이상 | 사망 후 24시간 내 재시작 비율 |
| 빌드 다양성 | 5종 이상 | 주간 상위 빌드 분포 |
| D7 리텐션 | 30% 이상 | 7일차 잔존율 |

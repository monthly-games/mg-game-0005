## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 기본 유물/적/스킬 풀 안정화 |
| 7~10주 | 난이도/보상 곡선 튜닝 |
| 11~12주 | 첫 시즌 테마 준비 |

**핵심 과제:**
- 초기 빌드 밸런스 확립
- 사망 지점 분석 기반 난이도 조정
- 재도전 동기 부여

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 독 시즌: 독 관련 유물/빌드 테마 |
| 5개월 | 화염 시즌: 화염 관련 유물/빌드 |
| 6개월 | 주간 도전 모드 시작 (공통 룰/랭킹) |
| 7개월 | 냉기 시즌: 냉기 관련 유물/빌드 |
| 8개월 | 번개 시즌 + 여름 이벤트 |

**핵심 과제:**
- 시즌별 테마 이벤트 사이클 정착
- 주간 도전 모드 효과 검증
- 빌드 다양성 확대

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 메타 progression 확장 (새 챕터) |
| 10개월 | 엔딩 변주 추가 |
| 11개월 | 하드코어 모드 추가 |
| 12개월 | 1주년 이벤트 + 스코어 챌린지 |

**핵심 과제:**
- 고난이도 콘텐츠 추가
- 메타 진행 콘텐츠 확장
- 하드코어 유저 만족도
